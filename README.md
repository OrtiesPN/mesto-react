# mesto-react
## Проектная работа №10
Создано приложение с помощью библиотеки React в рамках которого перенесена основная функциональность проекта Mesto. Приложение рендерится с помощью отдельных компонентов и полуает базовые данные от сервера, а именно: информацию о пользователе, массив карточек с количеством лайков. Реализовано открытие-закрытие попапов.

## Проектная работа 11
Реализована работа попапов и соответствующих форм с помощью отдельных компонентов. Каждый инпут и форма валидируются общим хук-компонентом. Реализована возможность изменять данные пользователя, добавлять, ставить лайки и удалять карточки. 
В планах: добавить для каждой формы стэйт отправки, загрузчик и реализовать запрет на спам отправку запросов.
